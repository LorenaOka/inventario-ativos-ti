Para rodar esse programa:

1) Configure o node:

	- npm install

2) Rode a API:

	- npm start

Se tudo estiver configurado corretamente, você verá a mensagem:
    
	
	Servidor rodando em http://localhost:3000


3) Depois, abra OUTRO terminal (Ctrl + Shift + `) e rode o CLI:

	- npm run start-cli

E a interface deve abrir!
